# SmartInventory Pro

A modern desktop inventory management application built with Python & Tkinter.

## Features
- Low stock alerts
- Dark mode
- Inventory history tracking
- CSV export & printable reports
- Backup & restore system

## Requirements
- Python 3.9+

## Run
```bash
python Inventory_App.py
